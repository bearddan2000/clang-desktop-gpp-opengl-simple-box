# clang-desktop-gpp-opengl-simple-box

## Description
A POC for opengl, graphics library, in c.

## Tech stack
- clang
    - g++
    - mesa-common-dev

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
